# Business-gated MVP governance vertical slice

## Decision

Build the smallest standalone TEMPO repository that proves an upstream
commercial decision can mechanically gate bounded agent implementation.

The submission track is **Work & Productivity**. The primary user is a product
or innovation lead supervising coding-agent work; research, finance,
engineering, founders, and agencies are collaborating roles. It is not merely
a developer guardrail.

## Bootstrap boundary

The runtime gate does not exist at repository creation time. The direct human
instruction is therefore recorded separately in
`governance/bootstrap-authorization.json`. That instruction authorizes building
the gate, but cannot be reused as a sample or downstream MVP warrant.

## Verification plan

Run focused unit tests, the complete conformance suite, the credential-free demo,
the README command runner, selfcheck, and CI configuration checks. Receipts must
be generated by tooling rather than typed into this task.
